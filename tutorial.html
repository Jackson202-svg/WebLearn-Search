How To Make A Website Using HTML, CSS & JavaScript

Step 1: Create Your Project Folder

Your website structure should look like this:


/website-project
│── index.html
│── style.css
│── script.js

Step 2: Create index.html

This is the main structure of your website:


<!DOCTYPE html>
<html>
<head>
    <title>My Website</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>

    <h1>Hello World</h1>
    <button onclick="sayHello()">Click Me</button>

    <script src="script.js"></script>

</body>
</html>

Step 3: Add style.css

This controls how your website looks:


body {
    font-family: Arial, sans-serif;
    text-align: center;
    background-color: #f4f6f9;
}

h1 {
    color: #2563eb;
}

button {
    padding: 10px 20px;
    border: none;
    background-color: #2563eb;
    color: white;
    cursor: pointer;
}

Step 4: Add script.js

This makes your website interactive:


function sayHello() {
    alert("Welcome to my website!");
}

How It Works Together

• HTML builds the structure • CSS styles the design • JavaScript adds interactivity

When combined correctly, you create a fully functional website.
